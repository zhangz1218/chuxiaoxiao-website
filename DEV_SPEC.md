# 网站开发上线规范

## 上线前必须检查项
1. **第三方依赖确认**：如giscus评论、统计等功能，必须确认已经完成配置（安装、授权等）再上线，避免报错
2. **文字层级规范**：页面同一块说明文字的大小变化不能超过两级，避免视觉跳变
3. **规划功能同步**：已对外提及的规划功能如果未开发，要说明是「待开发」

## 文字层级规范
### 页面文字
- 主标题：text-2xl / text-3xl
- 副标题/说明文字：text-base 或 text-lg
- 辅助说明/小字：text-sm
- **不得直接从text-lg跳到text-xs**

### 动态页面文字
- 标题：text-4xl / font-bold
- 副标题说明：text-base
- 卡片说明文字：text-sm
- 辅助备注：text-xs（仅最次要提示信息）
